Redesign Project

Conte's Pizza

I chose Conte's Pizza in Princeton, New Jersey for our website redesign project.

https://contespizzaandbar.blog/

Usability Problems

- Redundant links 
- Menu bar scrolls away
- Food menu is a static image
- Contact number is hidden in wall of text
- No indication as to how to order
- There's two titles with "Conte's" right next to each other
- No clear indication to return to Main Page

Accesibility Problems

- HTML elements are read aloud in screenreader
- Invisible elements that aren't used
- No Main/Home page button that's clearly indicated as being such (instead it's a clickable image)
- Two instances of same complicated map widget on the same page
- Title of restaurant is a paragraph while some regular information uses header tags
- Only one language option
